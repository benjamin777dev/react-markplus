# editor.markdown

A lightweight markdown editor.

![editor.markdown](https://github.com/tylerlong/editor.markdown/raw/master/editor.markdown.png)


## Setup & Run

```shell
git clone git@github.com:tylerlong/editor.markdown.git && cd editor.markdown
bower install
open index.html
```


## 如何嵌入到自己的项目

参考`index.html`，比葫芦画瓢。
