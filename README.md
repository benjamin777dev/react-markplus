# Markdown Plus

Markdown editor with extra features.

![Markdown Plus](https://github.com/tylerlong/markdown-plus/raw/master/screenshot.png)


## Setup & Run

```shell
git clone git@github.com:tylerlong/markdown-plus.git && cd markdown-plus
bower install
open index.html
```
